WaW MPData → Plutonium unlockstats_mp

Offline converter for Call of Duty: World at War PC multiplayer profiles. It decrypts a retail mpdata locally, verifies both integrity layers, and downloads only the 8,192-byte unlockstats_mp import block used by current Plutonium T4 builds.

1. Select the player file

Supported sizes: retail encrypted container 0x211c (8,476 bytes), decrypted payload 0x2104 (8,452 bytes), and raw Plutonium block 0x2000 (8,192 bytes). The CD key is used only in this browser tab and is cleared after a successful decryption.

3. Save the Plutonium file

Save this extensionless file as unlockstats_mp. The current import location confirmed from the supplied T4 folder is %LOCALAPPDATA%\Plutonium\storage\t4\plutonium\unlockstats_mp. This is an import template, not the profile's live encrypted mpdata; activate the intended multiplayer profile and run /unlockall once after installing it.
The converter copies the entire player-statistics block, including rank, prestige, totals, unlocks, challenges, and class/loadout data. Custom class names do not transfer to Plutonium and must be renamed in-game after import.

No network requests are made. The selected file, decrypted profile, and CD key remain on this computer. GPL-2.0-or-later; IWM support is based on codmpdatadec by Luigi Auriemma (2009). No warranty.
